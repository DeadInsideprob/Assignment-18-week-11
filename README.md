# Lab Assignment: 18
This program reads student grades from a text file and displays them on the screen. In the case the text file is missing there is a try-catch block.

## How to run
1. Open the program in your IDE.
2. Create the text file by right-clicking the file ad creating a new text file
3. Name the text file to be grades.txt
4. Enter the names of the students and scores in the text file.
5. View the output of the text file.

## Example output
Student grades:
John: 85
Sara: 92
Mark: 78

## Example of input in grades.txt
John: 85
Sara: 92
Mark: 78
